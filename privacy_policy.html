대디업 개인정보 처리방침

시행일: 2025년 1월 29일

Studio188(이하 "회사")은 대디업 애플리케이션(이하 "앱")을 운영하며, 이용자의 개인정보를 중요하게 생각하고 「개인정보 보호법」을 준수하고 있습니다.

제1조 (수집하는 개인정보 항목)

회사는 서비스 제공을 위해 다음과 같은 개인정보를 수집합니다:

1. 필수 수집 항목

  • 출산 예정일
  • 아기 이름 (태명 포함)
  • 아기 성별

2. 자동 수집 항목

  • 앱 사용 기록 (체크리스트 완료 상태)
  • 테마 설정 정보
  • 알림 설정 정보
  • 광고 식별자 (ADID/IDFA) - Google AdMob 광고 제공 목적
중요: 모든 개인정보는 사용자의 기기에만 저장되며, 별도의 서버로 전송되지 않습니다.

제2조 (개인정보의 수집 및 이용 목적)

수집된 개인정보는 다음의 목적으로 사용됩니다:

  • 맞춤형 임신 정보 및 주차별 가이드 제공
  • D-Day 카운트다운 기능 제공
  • 주차별 체크리스트 제공
  • 알림 서비스 제공
  • 테마 커스터마이징 제공
  • 광고 표시 (Google AdMob)

제3조 (개인정보의 보유 및 이용 기간)

개인정보는 다음과 같이 보유 및 이용됩니다:

  • 보유 위치: 사용자의 기기 내부 저장소 (SharedPreferences)
  • 보유 기간: 앱 삭제 시까지
  • 삭제 방법: 앱 삭제 시 자동 삭제

제4조 (개인정보의 제3자 제공)

회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다:

1. Google AdMob (광고 서비스)

  • 제공 항목: 광고 식별자(ADID/IDFA), 앱 사용 통계
  • 제공 목적: 맞춤형 광고 제공
  • 보유 기간: Google 개인정보 처리방침에 따름
  • 자세한 내용: Google 개인정보 처리방침

2. Firebase Analytics (선택적)

  • 제공 항목: 앱 사용 통계, 이벤트 데이터
  • 제공 목적: 서비스 개선을 위한 분석
  • 보유 기간: Firebase 정책에 따름

제5조 (개인정보의 안전성 확보 조치)

회사는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다:

  • 개인정보는 사용자 기기에만 저장되며 외부 서버로 전송되지 않음
  • 안드로이드 보안 샌드박스 내에서 데이터 보호
  • 암호화된 SharedPreferences 사용

제6조 (이용자의 권리와 행사 방법)

이용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

  • 개인정보 열람 요구: 앱 내 설정에서 확인 가능
  • 개인정보 수정: 앱 내 설정에서 직접 수정 가능
  • 개인정보 삭제: 앱 삭제 또는 앱 내 데이터 초기화
  • 광고 추적 제한: 기기 설정에서 광고 ID 재설정 또는 추적 제한

제7조 (아동의 개인정보 보호)

회사는 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 본 앱은 성인(예비 부모)을 대상으로 합니다.

제8조 (개인정보 처리방침의 변경)

이 개인정보 처리방침은 법령이나 서비스의 변경사항을 반영하기 위해 수정될 수 있습니다.

  • 변경 시 앱 내 공지 또는 업데이트 노트를 통해 안내
  • 중요한 변경사항은 시행 7일 전 공지

제9조 (개인정보 보호책임자)

개인정보 보호책임자

  • 회사명: Studio188
  • 담당자: 개인정보 보호 담당자
  • 이메일: studio188.help@gmail.com
  • 앱 내 문의: 설정 > 문의하기

제10조 (권익침해 구제방법)

개인정보 침해에 대한 신고나 상담이 필요한 경우 아래 기관에 문의하실 수 있습니다:

  • 개인정보보호 포털 (privacy.kisa.or.kr / 국번없이 118)
  • 개인정보 침해신고센터 (privacy.kisa.or.kr / 118)
  • 대검찰청 사이버수사과 (www.spo.go.kr / 1301)
  • 경찰청 사이버수사국 (ecrm.police.go.kr / 182)
